# DevOps Bootcamp Project

Welcome to the DevOps Bootcamp Project! This repository is designed to provide hands-on learning experiences across various essential DevOps domains, including bash scripting for system administration, Docker for containerization, and Kubernetes for container orchestration.

Each module is structured as a comprehensive course, complete with learning objectives, practical tasks, and homework assignments to solidify your understanding.

## Modules

### 1. Bash Scripting for Log Analysis
*   **Objective:** Master bash scripting for efficient log analysis and system administration tasks.
*   **[Go to Bash Module](./bash/Readme.md)**

### 2. Docker & Containerization
*   **Objective:** Understand Docker fundamentals, build custom images, and deploy multi-service applications with Docker Compose.
*   **[Go to Docker Module](./docker/Readme.md)**

### 3. Kubernetes: From Concepts to Deployment
*   **Objective:** Learn to deploy, manage, and scale containerized applications on Kubernetes, covering networking, storage, and advanced deployment patterns.
*   **[Go to Kubernetes Module](./kubernetes/Readme.md)**

---

## Getting Started

To begin, navigate to the `bash/`, `docker/`, or `kubernetes/` directories and follow the `Readme.md` instructions within each module.

## Prerequisites

Ensure you have the following tools installed on your system:

*   Git
*   Bash shell
*   Docker and Docker Compose (for the Docker module)
*   kubectl and access to a Kubernetes cluster (for the Kubernetes module)
*   `jq` and `yq` (for advanced bash log analysis)

Happy learning!
